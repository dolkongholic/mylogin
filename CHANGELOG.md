# 업데이트 내역 (Changelog)

MangoLogin 버전별 변경 사항입니다. 최신 버전이 맨 위에 옵니다.

---

## v0.2.0 — 라벨 · 동기화 수정

- 🏷️ **라벨(다중)**: 한 항목에 여러 라벨을 지정할 수 있습니다 (기존 단일 카테고리 대체). 기존 카테고리는 라벨로 자동 이전됩니다.
- 🔄 **동기화 가져오기 버그 수정**: 재로그인·다른 기기·재설치 후에도 같은 마스터 비밀번호면 서버 항목이 정상적으로 내려옵니다. (서버에 저장된 salt 기준으로 계정 키를 파생하도록 변경)
- 🌐 **URL 파비콘**을 기본 아이콘으로 표시 (DuckDuckGo)
- 📐 그리드 카드 높이 통일
- 🔒 자동 잠금(미사용 시 1/5/10/20분), 🔡 글자 크기 조절, ↕️ 리스트/그리드 드래그 정렬
- ⬆️ **업데이트 알림 개선**: 다운로드가 끝나면 재시작 여부를 팝업으로 물어보고, 업데이트 후 첫 실행 시 패치 내용을 보여줍니다.

## v0.1.0 — 최초 버전

- 로컬 우선 암호화 금고(AES-256-GCM, 마스터 비밀번호)
- mangomail 계정 기반 서버 동기화 (제로지식)
- 비밀번호 생성기, 즐겨찾기, 검색
- 커스텀 타이틀바 / 다크·라이트 테마 / SVG 아이콘
