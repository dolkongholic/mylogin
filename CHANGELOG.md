# 업데이트 내역 (Changelog)

MangoLogin 버전별 변경 사항입니다. 최신 버전이 맨 위에 옵니다.

---

## v0.3.8 — 공유 표시 변경

- ⚪ 배경 틴트/좌측선 제거, **작은 원형 점**으로 공유 구별 (받음=파랑, 내가 공유=초록). 리스트는 끝, 그리드는 우상단.

## v0.3.7 — 공유 구별 · 커서 제거

- 🖱️ 리스트 끝 마우스 커서 아이콘 제거.
- ↙️ 공유 방향 구별: 받은 항목 "받음"(파랑 배경), 내가 공유한 항목 "공유중"(초록 좌측선).
- 🔤 파비콘 없으면 선택 아이콘 → 없으면 제목 첫 글자.

## v0.3.6 — 실제 반영 수정 (중요)

- 🛠️ **배포 버그 수정**: 그동안 릴리스 시 렌더러(화면)를 재빌드하지 않아 옛 화면이 배포되던 문제 수정 (`release` 스크립트로 electron-vite build 항상 수행). 이제 변경이 실제로 반영됩니다.
- © 하단 우측 "© 2026. NTPERCENT" 표시.

## v0.3.5 — 동기화 안정화

- 🔔 토스트 최대 2개 제한 + 중복 방지. 공유 자동 새로고침은 조용히 처리(스팸 제거).
- 🔒 동기화 작업 직렬화(동시 실행 방지).
- ⬆️ 전체 업로드는 현재 로컬 상태만 업로드하고 되받아 내려오지 않음.
- 🗑️ 공유한 항목 삭제 시 서버 공유도 정리 → 삭제 항목이 폴링으로 되살아나지 않음.
- 🔓 서버 동기화 로그인 유지(토큰 저장).

## v0.3.4 — 공유 그룹/표시 정리

- 🗂️ 받은/보낸 공유를 하나의 **"공유" 라벨**로 묶어 표시. 공유 해제 시 수신자 목록에서 자동 제거(자동 새로고침).
- 🎨 공유받은 항목 옅은 테두리/배경 구별.
- 🏷️ 리스트 라벨 최대 3개 + `+N` 축약, 세로 정렬 정리.
- 🖱️ 리스트 끝 아이콘을 "더블클릭하여 열기" 표시로 변경.
- © 저작권 표기 **© 2026. NTPERCENT**.

## v0.3.3 — 로그인 유지 · 공유 자동 반영

- 🔓 서버 동기화 **로그인 유지**: 업데이트/재시작 후에도 로그인 상태 유지 (토큰 저장).
- 🔁 **공유 자동 반영**: 공유한 항목을 수정하면 수신자에게 자동 재공유되고, 수신자 앱은 20초 주기로 자동 새로고침.
- 🎨 공유받은 항목 **옅은 테두리/배경**으로 구별.
- 🏷️ 리스트 보기에서 **라벨 전부 표시**.

## v0.3.2 — 받은 공유 확인

- 서버 동기화 창에 "받은 공유" 버튼(개수/복호화 실패 표시).

## v0.3.1 — 리스트에서 즐겨찾기

- ⭐ 리스트/그리드 카드의 별을 눌러 **즐겨찾기를 바로 토글** (상세창에 들어가지 않아도 됨).
- 하단 상태줄 "업데이트 확인"/"기록" 버튼, 상세창 "공유" 버튼 재확인.

## v0.3.0 — 항목 공유 (종단간 암호화)

- 🤝 **항목 공유**: 상세 창의 "공유" 버튼으로 mangomail 계정(이메일)에게 항목을 공유.
- 🔐 **E2E**: 상대의 공개키(Curve25519)로 암호화해 전달 — 서버(mangomail)도 내용을 볼 수 없음.
- 👀 **권한**: "보기" / "수정 가능" 선택. 수정 가능 시 받은 사람이 고치면 소유자에게 반영.
- 🏷️ 받은 항목 = "공유받음" 라벨, 보낸 항목 = "공유함" 라벨 자동.
- 공유는 서버 동기화 로그인 상태에서 동작합니다.

## v0.2.5 — 트레이 안내 · 자동 실행

- 🗔 닫기 = 트레이 상주(창 숨김). 처음 닫을 때 **트레이 위치 안내 풍선** 표시 (Windows 11은 새 트레이 아이콘을 “^” 숨은영역에 넣음).
- 🚀 설정에 **"PC 시작 시 자동 실행"** 옵션 추가.

## v0.2.4 — 하단 상태줄 버튼

- ⬇️ 하단 상태줄에 **"업데이트 확인"** 링크 버튼 추가.
- 🕘 하단 상태줄 **"기록"** 버튼으로 패치 내용 팝업을 다시 볼 수 있습니다.

## v0.2.3 — 새 소식 알림 정상화

- 🎉 업데이트 후 첫 실행 시 **"새 소식(What's New)" 창이 확실히 표시**되도록 로직 수정 (표시 기록 키 갱신).

## v0.2.2 — 패치 내용 보기

- 🆕 **설정 → "패치 내용 보기"** 버튼: 이번 버전의 새 기능을 언제든 다시 볼 수 있습니다. (업데이트 후 첫 실행 시 자동으로도 표시)

## v0.2.1 — 트레이 상주

- 🗔 **닫기 = 트레이로 최소화** (종료 안 함). 트레이 아이콘 클릭으로 다시 열기, 우클릭 "종료"로 완전 종료.
- ⬆️ 자동 업데이트 서버 연결 완료 — 이 버전부터 새 버전이 자동으로 확인·적용됩니다.

## v0.2.0 — 라벨 · 동기화 수정

- 🏷️ **라벨(다중)**: 한 항목에 여러 라벨을 지정할 수 있습니다 (기존 단일 카테고리 대체). 기존 카테고리는 라벨로 자동 이전됩니다.
- 🔄 **동기화 가져오기 버그 수정**: 재로그인·다른 기기·재설치 후에도 같은 마스터 비밀번호면 서버 항목이 정상적으로 내려옵니다. (서버에 저장된 salt 기준으로 계정 키를 파생하도록 변경)
- 🌐 **URL 파비콘**을 기본 아이콘으로 표시 (DuckDuckGo)
- 📐 그리드 카드 높이 통일
- 🔒 자동 잠금(미사용 시 1/5/10/20분), 🔡 글자 크기 조절, ↕️ 리스트/그리드 드래그 정렬
- ⬆️ **업데이트 알림 개선**: 다운로드가 끝나면 재시작 여부를 팝업으로 물어보고, 업데이트 후 첫 실행 시 패치 내용을 보여줍니다.

## v0.1.0 — 최초 버전

- 로컬 우선 암호화 금고(AES-256-GCM, 마스터 비밀번호)
- mangomail 계정 기반 서버 동기화 (제로지식)
- 비밀번호 생성기, 즐겨찾기, 검색
- 커스텀 타이틀바 / 다크·라이트 테마 / SVG 아이콘
